# MapWar

A 2D turn-based strategy game built with **Phaser 3** and **TypeScript**.

## Features

- **Turn-Based Strategy**: Plan your moves carefully to expand your territory.
- **Planning Phase**: Select multiple cells to conquer in a single turn.
- **Economy & Combat**:
  - **Capture**: 10G per cell.
  - **Attack**: 20G for adjacent enemies.
  - **Distance Penalty**: 40G for attacking enemies far from your established territory.
- **Victory Condition**: Capture the enemy base to win!

## Development

### Setup

```bash
npm install
```

### Run Locally

```bash
npm run dev
```

### Run Tests

```bash
npm test
```

## Tech Stack

- **Engine**: Phaser 3
- **Language**: TypeScript
- **Bundler**: Vite
- **Testing**: Vitest
